Скрипт предназначен для создания отчетов ДД в НЦЗ, dd_post, ДД ИнтерРАО в НЦЗ, Факт ИнтерРАО в НЦЗ, VC_PC в соответствующих каталогах, и отправки письма с необходимыми отчетами в отдел мощности.
Перед выполнением скрипта необходимо задать параметры для отчета в файле config.py (открывать можно в блокноте).
Для корректной работы скрипта в соответствующей папке "Реестр ДД в НЦЗ" быть реестр ДД, а в базе дожны быть соответствующие данные.
Скрипт проще запускать через командную строку с помощью команды:
python c:\report_intrao\mail.py
где 
python - переменная среды python
c:\report_intrao\mail.py - путь к файлу скрипта
Можно создать отчеты, но не отправлять письмо, для этого необходимо выполнить команду:
python c:\report_intrao\report_intr.py
